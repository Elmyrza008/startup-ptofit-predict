🚀 Startup Profit Predictor

Startup Profit Predictor — это интеллектуальное веб-приложение, которое использует методы машинного обучения для прогнозирования чистой прибыли стартапа на основе его ключевых операционных расходов.

Приложение анализирует инвестиции в исследования и разработки (R&D), административные расходы и маркетинг, после чего мгновенно рассчитывает ожидаемую прибыль компании.

⚙️ Основные возможности
📊 Прогнозирование прибыли

Пользователь вводит данные о расходах стартапа (R&D, Administration, Marketing), после чего модель машинного обучения мгновенно рассчитывает ожидаемую прибыль, основываясь на обученных коэффициентах.

📈 Визуализация данных

Приложение автоматически генерирует регрессионный график, демонстрирующий математическую зависимость между затратами и потенциальной прибылью.

🗂 Предпросмотр датасета

В интерфейсе доступна таблица Dataset Preview, позволяющая изучить реальные исторические данные, на которых обучалась модель.

⚡ Мгновенное взаимодействие

Данные отправляются на сервер без перезагрузки страницы, что делает взаимодействие быстрым и удобным.

🧠 Используемый алгоритм

Проект основан на алгоритме Multiple Linear Regression (множественная линейная регрессия).

Модель анализирует влияние каждой категории расходов и определяет их вес в формировании прибыли, используя корреляции, выявленные в исторических данных о стартапах.

🛠 Технологический стек
Backend (Python)

Flask — легкий веб-фреймворк для обработки HTTP-запросов

Scikit-learn — построение и использование модели машинного обучения

Pandas — обработка и анализ CSV-датасета

NumPy — математические вычисления

Joblib — сохранение и загрузка обученной модели

Matplotlib — генерация графиков (PNG / Base64)

Frontend

HTML5 & CSS3 — кастомный интерфейс с премиальным темным дизайном и золотыми акцентами

JavaScript (Fetch API) — асинхронная отправка данных на сервер

Google Fonts (Montserrat) — современная и минималистичная типографика

🔄 Как работает система

1️⃣ Пользователь вводит данные о расходах стартапа
2️⃣ Данные отправляются на сервер через Fetch API
3️⃣ Backend передает значения в обученную модель машинного обучения
4️⃣ Модель рассчитывает прогнозируемую прибыль
5️⃣ Результат и график регрессии отображаются в интерфейсе

💡 Startup Profit Predictor демонстрирует практическое применение машинного обучения в бизнес-аналитике и может служить основой для более сложных систем прогнозирования финансовых показателей.
